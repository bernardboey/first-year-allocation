ASAP

Automated Suite Allocation Program

for Yale-NUS First-Years

Version 0.2.0 (05 July 2021)

To begin, please upload a .csv file containing information about the first-years. Each row (except the first) should represent a student. The first row should contain headers. There should be columns representing the following:

  • Student ID (Student names can be used instead but must be unique)
  • School name / School ID
  • Sex (must be "M" or "F")
  • Country (e.g. "Singapore", "United States", etc.)
  • Accessibility (must be "Yes" or "No")
  • Available RCs (e.g. "Saga, Elm, Cendana")
  • Living Preferences (Each question must be multiple choice, and must be ordinal)
  • Other columns are allowed, and they will not interfere with the allocation.

There must be no missing values. Please make sure your internet connection is on, as the program requires access to an online CSS stylesheet (Bootstrap). This will be fixed in future versions.
