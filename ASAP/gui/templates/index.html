ASAP

Automated Suite Allocation Program

for Yale-NUS First-Years

To begin, please upload a .csv file containing information about the first-years. Each row (except the first) should represent a student. The first row should contain headers. There should be columns representing the following:

  • Name/Student ID
  • School name or ID
  • Sex (must be "M" or "F")
  • Country (e.g. "Singapore", "United States", etc.)
  • Living Preferences (Each question must be multiple choice, and must be ordinal)
